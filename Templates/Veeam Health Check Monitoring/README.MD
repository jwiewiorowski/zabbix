Veeam Health Check Monitoring (Zabbix Template)
Overview

This template is designed to monitor Veeam Backup & Replication Health Check errors using Windows Event Log integration in Zabbix.

It detects specific error events generated by Veeam Backup & Replication and raises alerts when issues occur during health check operations.

Features
Monitors Veeam Backup Event Log
Detects Health Check errors (Event ID: 41710)
Uses active Zabbix agent
Generates triggers based on recent errors (last 10 hours)
Supports manual alert closure

Item Configuration
Item: healthcheck_error
Parameter	Value
Type	ZABBIX_ACTIVE
Key	eventlog[Veeam Backup,,"Error",,41710,,skip]
Value Type	LOG
History	7 days
Trends	Disabled
Description

This item collects error-level events from the Veeam Backup event log with:

Event ID: 41710
Severity: Error

These events typically indicate Health Check failures in backup jobs.

Trigger Configuration
Trigger Name
Server ({HOST.NAME}): Veeam problem Healthcheck ( {ITEM.VALUE} )
Expression
find(/Veeam check Healthcheck/eventlog[Veeam Backup,,"Error",,41710,,skip],10h)=1
Severity
AVERAGE
Manual Close
Enabled
Description

The trigger fires when at least one matching error event is found within the last 10 hours.
